 The WP Symposium Pro Profile Widget is used in conjunction WP Symposium Pro Plugin. It allows you to display user information and links and by adding additional shortcodes you can display just about anything in a themes sidebar.


WP Symposium Pro Sidebar Widget for user info display

== Description ==

The Widget allows you to setup your profile on you WP Symposium Pro Social Network Site.  The widget makes it easy to add and remove content from the WP Symposium Pro widget without having to code a text widget. Features in this widget include:

Official Release Version 14.8.4 – Now available.

   1. Widget Title, Avatar, friends, menu and 4 shortcodes 
      only show when user is logged in.
   2. Display Avatar – options:  
         1. No Avatar
	 2. 100 x 100px
	 3. 150 x 150px
	 4. 200 x 200px
	 5. 250 x 250px
	 5. 300 x 300px
   3. Display  5 Friends – options:  show or not
   4. Choose a User Menu
   5. Add 6 additional Shortcodes with Titles for each, 
	4 that show only when user is logged in
	2 that show in both states

== Installation ==

Go to plugins in your admin dashboard and click Add New. Choose upload and select the wp-symposium-pro-profile-widget.zip file.
